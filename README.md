# OIBSIP_PYTHON-PROGRAMMING_Task-2

Objective :

* To generate strong and random passwords
* To allow users to choose password length and characters
* To improve basic Python programming skills

Steps Performed :

* Get password length from the user
* Select character types (letters, numbers, symbols)
* Generate random characters using Python
* Combine characters to form a password
* Display the generated password

Tools Used :

* Python programming language
* Random module
* VS Code / IDLE
* Tkinter (for GUI version, if used)

Outcome in Brief :

* The program generates secure random passwords
* User-defined password criteria are supported
* Helps users create strong and safe passwords.
